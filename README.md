# Bayer Leverkusen 2023/24 – Invincible Season Analysis

This repository contains a football analytics project that examines Bayer Leverkusen’s **unbeaten Bundesliga season in 2023/24**.

The project uses **event-level data from StatsBomb** combined with additional league statistics to analyze the team's performance from tactical and statistical perspectives. Python scripts were used to process the data, compute key performance indicators (KPIs), and generate visualizations.

The analysis investigates several dimensions of Leverkusen’s success, including offensive patterns, build-up structure, defensive efficiency, pressing behavior, and late-match performance.

📄 Full explanations and interpretations of the results are available in the **PDF reports included in this repository**.
